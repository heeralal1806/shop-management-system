# How to Create a GitHub Account

## Step 1: Go to GitHub
1. Open your web browser
2. Go to: **://githuhttpsb.com**

## Step 2: Sign Up
1. Click the **"Sign up"** button (top right corner)
2. Enter your email address (e.g., heeralal8823@email.com)
3. Click **Continue**

## Step 3: Create a Password
1. Create a strong password (at least 8 characters)
2. Click **Continue**

## Step 4: Choose a Username
1. Enter a unique username (e.g., heeralal8823)
2. GitHub will tell you if it's available
3. Click **Continue**

## Step 5: Verify Email
1. GitHub will send a verification code to your email
2. Check your email and enter the code
3. Or click the verification link in the email

## Step 6: Complete Registration
1. Complete the human verification (if asked)
2. Click **Create account**

## Step 7: Customize Experience (Optional)
1. Answer a few questions about your experience level
2. Or skip this step
